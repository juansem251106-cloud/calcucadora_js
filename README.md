# Calculadora Web

## Descripción

Aplicación web de calculadora desarrollada con HTML, CSS y JavaScript. Permite realizar operaciones aritméticas básicas mediante una interfaz interactiva y fácil de usar.

## Funcionalidades

* Operaciones básicas: suma, resta, multiplicación y división
* Botón para limpiar la pantalla (C)
* Eliminación del último carácter (DEL)
* Visualización de la operación en tiempo real
* Diseño adaptable a diferentes tamaños de pantalla

## Funcionamiento

La aplicación responde a eventos de clic en los botones:

* `appendToDisplay(value)`
  Agrega valores (números u operadores) a la pantalla

* `clearDisplay()`
  Reinicia completamente la calculadora

* `deleteLast()`
  Elimina el último carácter ingresado

* `calculate()`
  Evalúa la operación matemática y muestra el resultado

---

## Ejecución

1. Clonar o descargar el repositorio
2. Abrir el archivo `index.html` en cualquier navegador web
3. Interactuar con la calculadora mediante los botones
